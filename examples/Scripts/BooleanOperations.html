Boolean Geometric Path Operations

Supported

  • All PathItem classes (Path and CompoundPath)
  • Boolean Union
  • Boolean Intersection
  • Boolean Subtraction
  • Resolving a self-intersecting Path

Not supported yet

  • Boolean operations between self-intersecting Paths
  • Paths that are clones of each other and lie in the exact same position
